# Youtube-Manager-App
A simple command-line YouTube video manager built in Python. It allows users to add, view, update, and delete video records with data stored in a JSON file for persistence.

# YouTube Video Manager (CLI)

A simple command-line application built in Python to manage YouTube videos.

## Features

- List all videos
- Add a new video
- Update video details
- Delete a video
- Data persistence using JSON file

## Technologies Used

- Python
- JSON (for storage)

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run the script:

```bash
python main.py
